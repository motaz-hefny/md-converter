# MoTekLab Proxy Suite - Distribution Hub

This repository serves as the official distribution channel for the **MoTekLab Proxy Suite**.

## 🚀 About the Application
MoTekLab Proxy Suite is a professional-grade proxy management, harvesting, and rotation engine designed for security research and data collection.

### Key Features:
- **Stealth Scraper**: High-performance engine for gathering proxies from public and private sources.
- **Smart Checker**: Validates proxy health, anonymity levels, and speed.
- **Proxy Server**: Local server with rotation and chaining support.
- **Privacy-First**: Strips identifying headers and blocks leaks.

## 🛠️ Installation
### Linux (AppImage)
1. Download the `.AppImage` file.
2. Right-click -> Properties -> Permissions -> **Allow executing file as program**.
3. Double-click to run.
