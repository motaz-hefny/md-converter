# MD Converter

A free, privacy-first bidirectional Markdown ↔ RichText converter. All processing happens 100% client-side in your browser.

**Live Demo:** [https://md.moteklab.com](https://md.moteklab.com)

## Features

### Core Functionality
- **Bidirectional Conversion** - Edit Markdown or RichText, both stay in sync
- **Real-time Preview** - See changes instantly as you type
- **Multiple Import Formats** - Markdown (.md), HTML (.html, .htm), RTF (.rtf)
- **Multiple Export Formats** - Markdown, HTML, PDF, DOCX, RTF
- **Privacy First** - No data collection, no storage, no AI training
